# password-strength-checker-
my first repository 
A Python CLI tool that checks password strength based on 
length,
character variety, 
common security rules
tells the user which portal the password is for.

WHAT IT DOES?
checks length, uppercase/lowercase, digits, special characters, spaces
Hides password input while typing
Gives a strength score and actionable suggestions
Loop to check multiple passwords

Tech Stack
Python 3, re, getpass (both standard library — mention no external dependencies needed).
HOW TO RUN?
Make sure Python 3 is installed.
Download or clone this repository.
Open the project in VS Code or PyCharm.
Run
